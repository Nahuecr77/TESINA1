STOCK DE COMPUTADORAS
-----------------------------------------------------------------------------------------------------------
este proyecto es desarrollar una página web que permita realizar un seguimiento detallado del stock de computadoras en el colegio, verificando tanto su disponibilidad como su uso. A través de un sistema automatizado, se podrá garantizar que las computadoras estén disponibles, se encuentren en buen estado de funcionamiento y sean fácilmente localizadas en los distintos espacios.
